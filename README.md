# 标签页会话管理器

一个简洁实用的 Edge/Chrome 浏览器扩展，帮助你保存、管理和恢复浏览器标签页会话。适合需要频繁切换工作场景的用户，一键保存当前工作环境，随时恢复。

## 功能特性

### 核心功能

- **一键保存** - 快速保存当前窗口所有标签页为一个会话
- **智能命名** - 自动生成会话名称（日期 + 时间 + 标签页数量）
- **会话恢复** - 在新窗口中一键恢复完整会话
- **数据持久化** - 使用浏览器本地存储，数据不丢失
- **数量限制** - 最多保存 50 个会话，避免存储溢出

### 会话管理

- **添加当前页面** - 将当前浏览的页面添加到已有会话中
- **快捷添加** - 每个会话卡片旁的「+添加」按钮，一键添加
- **详情预览** - 展开查看会话内的所有标签页
- **灵活编辑** - 支持修改会话名称、删除单个标签页
- **数量同步** - 删除或添加标签页后，会话名称中的数量自动更新

### 智能搜索

- **实时搜索** - 根据会话名称、网页标题或网址快速筛选
- **关键词高亮** - 匹配的关键词自动高亮显示（黄色背景）
- **自动展开** - 匹配标签页时自动展开对应会话
- **过滤显示** - 搜索时仅显示匹配的标签页，快速定位
- **匹配计数** - 显示「匹配 X 个标签页」，结果一目了然

## 安装说明

### 前置要求

- Edge 浏览器（Chromium 内核）或 Chrome 浏览器

### 安装步骤

1. **下载项目**

   ```bash
   git clone https://github.com/TheAceCream/TabSessionManager.git
   ```
   
   或直接下载 ZIP 压缩包并解压。

2. **打开扩展管理页面**

   - Edge: 访问 `edge://extensions/`
   - Chrome: 访问 `chrome://extensions/`

3. **启用开发者模式**

   在页面右上角开启「开发者模式」开关。

4. **加载扩展**

   点击「加载解压缩的扩展」按钮，选择项目文件夹。

5. **完成安装**

   扩展图标将出现在浏览器工具栏，点击即可使用。

## 使用方法

### 保存会话

1. 打开若干标签页
2. 点击扩展图标
3. 点击「保存当前窗口所有标签页」按钮
4. 会话自动保存，显示成功提示

### 添加当前页面

**方式一：通过顶部按钮**
1. 点击「添加当前页面到会话」按钮
2. 在弹出的会话列表中选择目标会话
3. 页面添加成功

**方式二：通过快捷按钮**
1. 找到目标会话卡片
2. 点击会话名称旁的「+添加」按钮
3. 当前页面立即添加到该会话

### 恢复会话

1. 点击扩展图标
2. 找到目标会话（可使用搜索框筛选）
3. 点击「全部恢复」按钮
4. 新窗口自动打开所有标签页

### 搜索会话

1. 在搜索框输入关键词
2. 系统自动筛选并显示匹配结果
3. 匹配的标签页会高亮显示，会话自动展开
4. 仅显示匹配的标签页，便于快速定位

### 管理会话

- **修改名称** - 点击会话名称即可编辑
- **查看详情** - 点击「展开详情」查看全部标签页
- **删除标签页** - 悬停标签页项，点击 × 按钮
- **删除会话** - 点击「删除」按钮（需确认）

## 技术栈

| 技术 | 说明 |
|------|------|
| Manifest V3 | Chrome 扩展最新规范 |
| Chrome Extensions API | tabs、storage、windows |
| JavaScript (ES6+) | 原生 JavaScript，无框架依赖 |
| CSS3 | 原生 CSS，无预处理器 |

### 核心依赖

- `chrome.tabs` - 获取和创建标签页
- `chrome.storage.local` - 本地数据持久化
- `chrome.windows` - 创建新窗口

## 数据存储

会话数据存储在 `chrome.storage.local`，结构如下：

```javascript
{
  sessions: [
    {
      id: "1704067200000",           // 唯一ID（时间戳）
      name: "2026/5/12 18:30（5个标签页）",
      createdAt: "2026-05-12T10:30:00.000Z",
      tabs: [
        {
          title: "页面标题",
          url: "https://example.com",
          favIconUrl: "https://example.com/favicon.ico"
        }
      ]
    }
  ]
}
```
